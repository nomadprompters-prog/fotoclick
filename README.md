# Fotosclick

Sitio web (Next.js) para el proyecto Fotosclick.

## Scripts útiles
- `npm run dev` — desarrollo
- `npm run build` — build de producción
- `npm run start` — sirve la build

## Estructura
- `src/` — código fuente Next.js (App Router)
- `public/` — estáticos públicos
- `scripts/` — utilidades

## Deploy
Se sirve desde el VPS con Docker/Traefik. Artefactos de build y backups están ignorados en Git (`.gitignore`).
